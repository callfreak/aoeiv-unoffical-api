Age of Empires IV Content Editor API

Welcome to the unofficial documentation for the Age of Empires IV Content Editor API. This documentation provides comprehensive information about working with texts and images in the Content Editor.

Getting Started

The Content Editor API allows you to programmatically manage game content including text localization and image assets. This API is designed for modders and developers who want to create custom content for Age of Empires IV.

Key Features

  • Text and localization management
  • Image asset handling and optimization
  • Batch operations for efficient content updates
  • Validation and error handling
