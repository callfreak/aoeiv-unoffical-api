# worldGetRandom
worldGetRandom()

## Description

This function returns a value **between 0.0 and 1.0**. It is **required for multiplayer matches** to ensure,
that all players recieve the **same random number**. Whithout using this function the game would crash.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| None | - | - |

## Returns

| Type | Description |
|------|-------------|
| Float | Random number between 0.0 and 1.0 |
