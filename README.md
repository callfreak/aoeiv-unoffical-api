This repository contains comprehensive documentation for working with the Age of Empires IV Content Editor.

## Documentation Site

Visit the documentation site at: [https://callfreak.github.io/aoeiv-unoffical-api/](https://callfreak.github.io/aoeiv-unoffical-api/)

## Local Development

To run the documentation site locally:

```bash
# Install dependencies
bundle install

# Run Jekyll server
bundle exec jekyll serve

```
