# Age of Empires IV Content Editor API Documentation

Unofficial documentation for the Age of Empires IV Content Editor API.

## About

This repository contains comprehensive documentation for working with the Age of Empires IV Content Editor API, focusing on text and image management capabilities.

## Documentation Site

Visit the documentation site at: [https://callfreak.github.io/aoeiv-unoffical-api/](https://callfreak.github.io/aoeiv-unoffical-api/)

## Features

- **Dark Mode UI**: Modern dark theme optimized for extended reading
- **Left Sidebar Navigation**: Easy access to all documentation categories
- **Comprehensive API Documentation**: Detailed information about Text and Image APIs
- **Code Examples**: Practical examples in JavaScript and Python
- **Interactive Reference**: Tab-based API reference with quick navigation

## Documentation Categories

- **Overview**: Introduction to the API architecture and core concepts
- **Text API**: Complete documentation for text and localization management
- **Image API**: Detailed guide for image asset handling and processing
- **Code Examples**: Working code samples for common use cases
- **API Reference**: Complete endpoint reference with tab navigation

## Local Development

To run the documentation site locally:

```bash
# Install dependencies
bundle install

# Run Jekyll server
bundle exec jekyll serve

# Visit http://localhost:4000/aoeiv-unoffical-api/
```

## Technology Stack

- **Jekyll**: Static site generator
- **GitHub Pages**: Hosting platform
- **Pure CSS**: Custom dark mode styling
- **Vanilla JavaScript**: Interactive features
